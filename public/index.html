TREC BioGen 2025 — Task B

Generate answers for 30 medical questions and compare systems.

Task Description (summary)

BioGen 2025 has two subtasks.

  • Task A — Grounding Answer: For each given answer sentence, add up to 3 new PMIDs that support it (prior supported PMIDs are provided) and up to 3 PMIDs that contradict it. PMIDs must come from the released PubMed snapshot.
  • Task B — Reference Attribution: For each topic, generate an answer (≤ 250 words). Each sentence may include up to 3 PMIDs from the released PubMed snapshot that support the statement.

Key dates: Dataset & baseline June 2, submissions Aug 15, results late Sep, papers late Oct, conference Nov 17–21.
